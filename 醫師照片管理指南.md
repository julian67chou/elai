# 伊萊診所醫師照片管理指南

## 當前醫師照片
- **URL**: https://images.unsplash.com/photo-1622253692010-333f2da6031d
- **描述**: 專業亞洲男性醫師，戴聽診器，適合心臟血管內科專家形象
- **尺寸**: 400x400像素 (響應式調整)

## 如何更新為真實照片

### 方法一：直接替換URL
1. 將施奕仲院長的專業照片上傳到圖床服務（如Imgur、Cloudinary）
2. 取得圖片URL
3. 編輯 `/workspace/elai-clinic/index.html` 檔案
4. 找到以下程式碼：
   ```html
   <img src="https://images.unsplash.com/photo-1622253692010-333f2da6031d..." 
        alt="施奕仲院長" class="doctor-img">
   ```
5. 將 `src` 屬性值替換為新的照片URL

### 方法二：使用本地檔案
1. 將醫師照片命名為 `doctor-shih.jpg` 並上傳到 `/workspace/elai-clinic/assets/` 目錄
2. 修改HTML中的圖片URL：
   ```html
   <img src="/assets/doctor-shih.jpg" alt="施奕仲院長" class="doctor-img">
   ```

### 照片規格建議
- **尺寸**: 最小 400x400 像素，建議 800x800 像素
- **格式**: JPG 或 PNG
- **背景**: 建議白色或專業背景
- **服裝**: 醫師袍或專業服裝
- **表情**: 專業、親切、自信

## 照片CSS樣式
醫師照片使用以下CSS樣式：
```css
.doctor-img {
    width: 150px;
    height: 150px;
    border-radius: 50%;      /* 圓形 */
    object-fit: cover;       /* 保持比例填滿 */
    margin-bottom: 20px;
    border: 5px solid #F8F4E9; /* 米色邊框 */
}
```
